# AI-Based Student Academic Advisor System

## Overview
This project is a Rule-Based Expert System designed to simulate academic advising using predefined rules.

## Features
- Forward-chaining inference engine
- Rule-based decision system
- Explainable AI output
- Hierarchical reasoning model

## Structure
- src/ → core AI system
- tests/ → test cases
- docs/ → documentation

## How to Run
```bash
python main.py
